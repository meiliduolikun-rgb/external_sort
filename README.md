# External sort

## Запуск

```text
python main.py input_file max_numbers
```

Пример:

```text
python main.py random_numbers.bin 1000000
```

`input_file` - бинарный файл без заголовка. В нем подряд записаны 32-битные целые числа.

`max_numbers` - максимальное количество чисел, которое можно одновременно загрузить в память.

Результат создается в той же директории:

```text
random_numbers.bin.sorted
```

## Алгоритм

Используется внешняя сортировка:

1. Файл читается частями.
2. Части сортируются параллельно через `multiprocessing.Pool`.
3. Отсортированные части записываются во временные бинарные файлы.
4. Временные файлы сливаются в один результат через `heapq`.

Сложность: `n * log(n)`.
